# Answer key to questions 

## Intro to SQL - Section 2.6.1

```sql
SELECT
	G.NAME ,
	ROUND(SUM(I.TOTAL), 2) AS SUM_TOTAL_GENRE
FROM 
	INVOICE I
LEFT JOIN 
	INVOICELINE IL 
ON
	I.INVOICEID = IL.INVOICEID
LEFT JOIN 
	TRACK T 
ON
	IL.TRACKID = T.TRACKID
LEFT JOIN 
	GENRE G
ON
	T.GENREID = G.GENREID
GROUP BY
	G.NAME
ORDER BY
	SUM(I.TOTAL) DESC;
```
